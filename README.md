## Admin LTE using Laravel + Vue 3

## Installation

<pre>
composer install
cp .env.example .env
php artisan key:generate
</pre>

## Install NPM Packages
<pre>
npm install
</pre>

## Run 
<pre>
php artisan serve
npm run watch
</pre>
